Reading notes & honesty
  • Data window starts 2021-01; the raw histories overlap 2019–2026 but רומן's earlier data is thin and uneven, so earlier rows are dropped for a fair baseline.
  • Genre is an external, partial add-on (web research); the long tail is bucketed as “Other” and always shown.
  • רומן has higher total volume, so “Absolute hours” vs “Normalized share” is a first-class toggle — normalized share keeps taste and routine comparisons fair.
  • Skip behavior uses the comparable under-30s measure — Apple and Spotify count native “skips” differently.
  • City-level location exists only for אור, so there is no map.
