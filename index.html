Reading notes & honesty
  • Data window starts 2021-01; earlier data is dropped for fairness (רומן's history is sparse before 2021).
  • Genre is an external, partial add-on; the long tail is bucketed as “Other” and always shown.
  • רומן has higher total volume, so the Absolute ↔ Normalized toggle is first-class: normalized share is what makes taste and routine comparisons fair.
  • Skip behavior uses the comparable under-30s measure — Apple and Spotify count native “skips” differently.
  • City-level location exists only for אור, so there is no map.
