About the data
  • The window starts 2021-01. The raw exports overlap 2019–2026, but רומן's data is thin before 2021, so the comparison starts where it's fair. University begins for both at 2024-01.
  • Genre tags are an external add-on (web research), collapsed into broad families — partial and provisional. What couldn't be named confidently stays in an honest “Other”.
  • “Shared” artists require real listening on both sides (at least one full 30-second play each), not just a name appearing twice.
  • Skips aren't compared — Apple and Spotify count them differently.
  • City-level location exists only for אור, so there is no map.
